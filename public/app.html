☁️ Cloud Sync

Each machine backs up its own agents, managers, tasks and event config to its own cloud namespace. Every machine runs its own schedules and responds to its own events independently.

Your config syncs automatically to this machine's own namespace. To borrow agents from another machine, use Browse Other Machines.

Advanced — manual sync

Sync happens automatically. Push forces an immediate backup of this machine's config to the cloud. Restore overwrites this machine's local config from its own latest cloud backup — use only for disaster recovery.
